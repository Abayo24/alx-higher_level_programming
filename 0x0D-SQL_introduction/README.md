# SQL Introduction

## Description
This project contains a series of SQL scripts for MySQL server, focusing on fundamental SQL concepts such as database creation, table manipulation, data querying, and more.

## Project Structure
- `0-list_databases.sql`: Script to list all databases on the MySQL server.
- `1-create_database_if_missing.sql`: Script to create a database if it doesn't exist.
- `2-remove_database.sql`: Script to delete a database if it exists.
- `3-list_tables.sql`: Script to list all tables in a database.
- ...

## Instructions
1. Install MySQL 8.0 on Ubuntu 20.04 LTS.
2. Connect to your MySQL server using the provided instructions.
3. Execute the SQL scripts using the `mysql` command-line client.

## Dependencies
- MySQL 8.0
- Ubuntu 20.04 LTS
